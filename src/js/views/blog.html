OneVue Application

End – End Document Automation with OneVue

Ecovue Solutions OneVue with Oracle Content Management Cloud and/or WebCenter Content is a centralized document management application portal to be able to process the document from Scanning to OCR to Verification to Exceptions Processing to Approvals and integrating with end ERP Applications like Oracle eBusiness Suite, Oracle PeopleSoft, SAP etc.

End – Key Features

  • Centralized User Interface: Ecovue OneVue Invoice Portal to process/manage all transactions for verifying/coding/exception processing/approval etc.
  • Dashboard: with Metrics, KPI’s, Charts
  • XML to PDF: For Italy eInvoicing
  • Excel Upload: Upload Invoices / Coding Rules from Excel
  • Auto Coding: User defined coding rules for auto coding based on multiple attributes
  • PO Matching: Match to PO, Match to Release, Match to Receipt with Custom PO Line Pairing
  • Federated Search: Users can search Invoices from within the Application and or from Content Repository
  • User Management: Optionally user directory can be maintained withing the application
